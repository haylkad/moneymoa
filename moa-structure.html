3 — 화면별 구성

🏠
오늘 (홈)
코치 브리핑(오늘 쓸 수 있는 돈·한마디·성적표) → 위시리스트 구매력 → 핵심 목표 → 총자산(실시간 추정+그래프) → 자산 구성
💳
가계부
월 이동 → 쓴 돈 + 고정/변동 칩 → 이 돈이면(변동지출 팩폭) → 저축률 → 고정지출 체크(✅🕐⚠️) → 카테고리 → 내역
📈
투자
평가금액·총수익 → 적립 투자(이번 달 입금·하루 환산·누적) → 종목별 수익률 (수익 빨강 / 손실 파랑)
🎯
목표
목표 카드: 게이지 · D-day · 이달 납입 · 페이스 예측(순항/딱 맞음/분발)
⚙️
설정
시트 URL 연결 · 목표 저축률(30~70%) · 시트 가이드 · 캐시 관리

4 — 핵심 계산식

총자산 (실시간 추정)
마지막 실측 스냅샷 + 이후 수입 − 이후 지출 · 저축은 현금→예적금 이동만
오늘 쓸 수 있는 돈
( 수입 × (1−저축률) − 고정비 − 이미 쓴 변동지출 ) ÷ 남은 일수
목표 현재금액
목표 시트 시작값 + 저축 탭 납입 합계 (자동 합산)
위시 구매력
이번 달 (수입−지출) ≥ 싼 것부터 누적합 → ✅ 표시
페이스 예측
목표 잔여액 ÷ 최근 3개월 평균 자산 증가액 → 예상 달성월 vs 마감 비교
