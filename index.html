구글시트 연결

Apps Script 웹앱 URL을 붙여넣으면 시트 데이터가 자동으로 들어와요. URL은 이 브라우저에만 저장돼요.

목표 저축률

번 돈에서 몇 %를 모을지 정하면, 코치가 매일 "오늘 쓸 수 있는 돈"을 계산해줘요.

처음 설정하기
1. 구글시트 만들기
새 스프레드시트에 탭 10개를 만들고, 1행에 아래 헤더를 그대로 입력하세요.
탭 이름1행 헤더
자산날짜 · 현금 · 예적금 · 국내주식 · 해외주식
수입날짜 · 항목 · 금액
지출날짜 · 카테고리 · 내용 · 금액
포트폴리오종목명 · 구분 · 수량 · 평단가 · 현재가
목표목표명 · 목표금액 · 현재금액 · 마감일
저축날짜 · 목표명 · 금액
고정지출항목 · 금액 · 결제일 · 카테고리 · 자동기록(Y/N)

위시리스트항목 · 가격
위시리스트에는 진짜 갖고 싶은 것만 적으세요. 이번 달 모은 돈으로 뭐까지 살 수 있는지 홈에서 보여줘요 — 작은 것부터 큰 것까지 가격대를 섞으면 더 재밌어요.
자산 탭은 월 1회 스냅샷 — 매달 말일에 한 줄씩 추가하면 추이와 페이스 예측이 정확해져요.
포트폴리오의 현재가는 =GOOGLEFINANCE("KRX:005930") 수식으로 자동 갱신, 해외는 =GOOGLEFINANCE("NASDAQ:NVDA")*GOOGLEFINANCE("CURRENCY:USDKRW")처럼 원화 환산해서 넣어주세요.
2. Apps Script 붙여넣기
시트에서 확장 프로그램 → Apps Script를 열고, 함께 받은 Code.gs 내용을 붙여넣으세요. 아래 버튼으로 복사해도 돼요.
3. 웹앱으로 배포하기
① 우측 상단 배포 → 새 배포
② 유형: 웹 앱
③ 액세스 권한: 모든 사용자 (URL을 아는 사람만 접근 가능해요)
④ 배포 후 나오는 /exec URL을 위 입력칸에 붙여넣기

시트를 수정한 뒤에는 앱 상단의 새로고침 버튼만 누르면 돼요. Apps Script 코드를 고친 경우에만 배포 관리 → 새 버전으로 다시 배포하세요.
